RESUMEN (Español)
OptiFincas es una solución integral diseñada para optimizar la administración de fincas. El sistema automatiza el procesamiento de extractos bancarios mediante algoritmos de clasificación y técnicas de reconocimiento de patrones para la categorización de movimientos y detección de inmuebles, permitiendo la edición manual y exportación de datos. Además, integra un algoritmo de optimización de rutas y horarios para el personal de limpieza, calculando desplazamientos eficientes por carretera mediante la integración de servicios de mapas en tiempo real. Actualmente, el desarrollo se centra en la generación automatizada de informes financieros detallados y la implementación de un portal interactivo para propietarios. El objetivo principal es digitalizar la gestión de comunidades, mejorando significativamente la eficiencia operativa y la transparencia en la administración de propiedades.

ABSTRACT (English)
OptiFincas is an integrated solution designed to optimize property management. The system automates bank statement processing using classification algorithms and pattern recognition techniques for transaction categorization and unit detection, enabling manual editing and data export. Furthermore, it integrates a route and schedule optimization algorithm for cleaning staff, calculating efficient road travel durations via real-time mapping services integration. Current development focuses on generating automated, detailed financial reports and implementing an interactive owner portal. The primary goal is to digitalize community management, significantly enhancing operational efficiency and transparency in property administration.
